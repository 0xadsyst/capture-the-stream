# Capture the Stream

## Installation

```bash
yarn
```


## Build and Deploy
```
yarn generate
```
Note: There will be an error the first time you run this, but this can be ignored.
``` 
yarn contracts:build
yarn deploy
yarn chain
yarn run-graph-node
yarn subgraph-build-deploy-local
yarn start:nextjs
```
Note that you'll need docker installed to run the graph node.
